## 1.19.0 (March 10, 2026)

NOTES:

* This release moves the [resource configuration generation](https://developer.hashicorp.com/terraform/language/import/generating-configuration) logic for the `-generate-config-out` flag from Terraform Core to the Framework for Terraform `v1.14.0` and above. There should be no functionality changes for resource configuration generation in this release. ([#1281](https://github.com/hashicorp/terraform-plugin-framework/issues/1281))
* This Go module has been updated to Go 1.25 per the [Go support policy](https://golang.org/doc/devel/release.html#policy). Any consumers building on earlier Go versions may experience errors. ([#1287](https://github.com/hashicorp/terraform-plugin-framework/issues/1287))
