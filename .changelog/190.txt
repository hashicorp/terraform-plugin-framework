```release-note:breaking-change
diag: The `DiagnosticWithPath` interface type has been merged into the `Diagnostic` interface type. If implementing custom `Diagnostic` types, the `Path()` and `SetPath()` methods are now required.
```

```release-note:breaking-change
diag: The `AttributeErrorDiagnostic`, `AttributeWarningDiagnostic`, `ErrorDiagnostic`, and `WarningDiagnostic` types has been merged into the `GenericDiagnostic` type, which can be embedded in any custom diagnostics type to simplify implementation.
```

```release-note:enhancement
diag: Added `Diagnostic` interface type `Path()` and `SetPath()` methods
```
